# LibraryManager (MVPC)
A sample Flask MVPC project for Library management connected to MySQL.

## Config
- Database: `libraryx`
- Host: `localhost`
- User: `root`
- Password: `123456`

## Run
1. Create venv and activate (optional)
2. `pip install -r requirements.txt`
3. Update DB config in `app/config/db_config.py` if needed
4. `python app/main.py`
